SPIDER LEAGUE

Connect the league database

These steps enable shared standings and restrict official submissions to your admin account. The setup stays on Firebase's no-cost Spark plan.

1. Create the Firebase project

  1. Open the Firebase console and select Create a project.
  2. Name it spider-league.
  3. Leave Google Analytics off; it is not needed.
  4. Remain on the no-cost Spark plan and do not enable billing.

2. Register the website

  1. On the project overview, select the Web icon (</>).
  2. Use Spider League Web as the app nickname.
  3. Do not enable Firebase Hosting; GitHub Pages remains the host.
  4. Select Register app and keep the displayed firebaseConfig object available.
  5. In GitHub, edit firebase-config.js and replace the four placeholder values with apiKey, authDomain, projectId, and appId.
  6. Commit the edit to main.

3. Create Firestore

  1. Open Build → Firestore Database and select Create database.
  2. Choose Standard edition and Production mode.
  3. Choose a nearby U.S. region; us-west1 is sensible for Portland when offered.
  4. Open the Rules tab.
  5. Copy all of firestore.rules from the GitHub repository into the editor and select Publish.

4. Create the admin login

  1. Open Build → Authentication and select Get started.
  2. Enable Email/Password under Sign-in method.
  3. Open Users, select Add user, and create your login.
  4. Copy the user's User UID.

5. Grant admin access

  1. Return to Firestore Database → Data.
  2. Create a collection named admins.
  3. Use the copied User UID as the document ID.
  4. Add a boolean field named enabled with the value true, then save.

6. Authorize the site

  1. Open Authentication → Settings → Authorized domains.
  2. Add jimmycrooks9-create.github.io if it is not already listed.

7. Test it

  1. Open admin.html and sign in.
  2. Add one test observation.
  3. Open league.html and confirm it appears.

Cost guardrail: do not attach a billing account. The Spark plan's no-cost Firestore allowance is far beyond the expected use of a small friend-group league.
