# Changelog

## 0.7.0 — Official league submissions

- Added a public ranked leaderboard with shared ranks for equal SDI scores.
- Added a protected admin page for approving and deleting official observations.
- Added automatic spider resolution and SDI snapshotting in the submission form.
- Added Firebase Authentication and Firestore integration scaffolding.
- Added strict public-read/admin-write Firestore rules.
- Added exact no-cost Firebase setup instructions.
- Added league ranking tests and Firebase diagnostics.

## 0.6.0 — 2026-06-27

- Added an in-app diagnostics page for local evidence files, iNaturalist, GBIF/Catalogue of Life, and service-worker support.
- Added a direct low-confidence Dysdera crocata score from eight verified bites.
- Added a low-confidence Kukulcania genus score and an explicit Kukulcania hibernalis species evidence gap.
- Added six common-name aliases and two fallback tests.
- Corrected the service-worker cache version.

## 0.5.0 — 2026-06-27

- Added cohort-table approximation rule that avoids double-counting overlapping severe-pain and systemic-effect groups.
- Added family fallbacks for Araneidae, Salticidae, Desidae, Corinnidae, and Zodariidae.
- Added a supported Steatoda genus score and an explicit Steatoda grossa species evidence gap.
- Expanded automated fallback tests.

## 0.4.0 — 2026-06-27

- Added shareable result URLs and native share/copy support.
- Added an evidence coverage page and Oregon-focused research queue.
- Expanded offline common-name and taxonomy coverage.
- Moved the broad white-tail common name to a supported Lampona genus record.

## 0.3.0 — 2026-06-27

- Added first Pacific Northwest SDI evidence records.
- Added explicit evidence-gap records for common regional spiders.
- Added JSON integrity validation and expanded unit tests.
- Added GitHub Actions testing and Pages deployment.
- Completed offline taxonomy snapshots for curated aliases.

## 0.2.0 — 2026-06-27

- Added PWA support, recent searches, evidence display, local taxonomy fallback, and core tests.
